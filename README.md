Docker CRIU Demo
================

An automated, annotated, interactive demo of live container migration using Virtualbox, Vagrant and ShutIt:

```
pip install shutit
git clone --recursive https://github.com/ianmiell/shutit-criu
cd shutit-criu
./run.sh
```
